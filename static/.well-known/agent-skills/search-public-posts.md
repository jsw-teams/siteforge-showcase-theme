# Search public posts

Use this skill to search public Blog.js.gripe posts.

## Tool

Call the WebMCP `search_public_posts` tool on `https://blog.js.gripe/`.

## Input

- `query`: search keywords.
- `locale`: optional locale, one of `zh-CN`, `zh-TW`, or `en`.
- `limit`: optional result limit from 1 to 30.

## Output

The tool returns matching post titles, descriptions, canonical URLs, dates, categories, and tags.
