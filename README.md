# Siteforge Showcase Theme

基于 Siteforge 二次开发的产品展示与产品文档主题示例。

## 直接使用

```bash
git clone https://github.com/jsw-teams/siteforge-showcase-theme.git
cd siteforge-showcase-theme
npm install
npm run check
npm run generate
npm run server
```

Local preview defaults to:

```text
http://127.0.0.1:4173/
```

Edit `config.yml`, `content/posts/`, `content/pages/`, and theme `source-assets/` first. Enter `themes/product-docs/` only when layout, styles, templates, or scripts need customization.
